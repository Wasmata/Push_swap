# 🔢 Push_swap

Projet de : 42 Perpignan consistant à trier une pile d'entiers avec un nombre minimal d’opérations.

## 🚀 Features
- Tri de nombres avec deux piles (`a` et `b`)
- Utilisation d’opérations spécifiques (`sa`, `pb`, `ra`, etc.)
- Optimisation du nombre de coups
- Gestion des erreurs (arguments invalides, doublons)

## ⚙️ Compilation & Usage
```bash
make
./push_swap 3 2 1
